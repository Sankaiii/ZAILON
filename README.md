# ZAILON — Universal Mod Launcher

> Local-first universal mod manager — by [@souanpt](https://github.com/Sankaiii)

## Download

Go to [Releases](https://github.com/Sankaiii/ZAILON/releases) and download for your platform.

## ⚠️ Windows SmartScreen Warning

When installing on Windows you may see a SmartScreen warning. This is **normal for open source apps without a paid code signing certificate**.

To install:
1. Click **"More info"**
2. Click **"Run anyway"**

The app is open source — you can read every line of code in this repo.

## Install

| Platform | File |
|---|---|
| Windows | `ZAILON_x64-setup.exe` |
| Linux | `zailon_amd64.AppImage` or `.deb` |
| macOS Apple Silicon | `ZAILON_aarch64.dmg` |
| macOS Intel | `ZAILON_x64.dmg` |

## Features
- 🎮 Universal game support
- 📦 Profile system
- 🧭 Explore mods — GameBanana, Nexus, CurseForge, AyakaMods
- ⚡ One-click install
- 🕒 Playtime tracking
- 🔒 Offline-first — no account, no cloud
